Resultado de Aprendizaje

Cristhian Nunez Jurado

En esta página, comparto el resultado de aprendizaje de la Unidad 1 de la materia de Aplicaciones Web I4.0. Durante esta unidad, hemos explorado y aplicado herramientas esenciales para el desarrollo web moderno.

Algunos de los aspectos destacados incluyen:

  • Aprendizaje y aplicación de Express, un marco de aplicación web para Node.js, que simplifica la creación de servidores web y el manejo de rutas y solicitudes HTTP.
  • Uso de Dotenv para una gestión segura de las variables de entorno en nuestras aplicaciones.
  • Desarrollo del frontend con Angular, un framework de código abierto que nos permite crear interfaces de usuario interactivas y modulares.

En resumen, esta unidad nos ha preparado para construir aplicaciones web de alto rendimiento y calidad, utilizando herramientas avanzadas y buenas prácticas de desarrollo.
